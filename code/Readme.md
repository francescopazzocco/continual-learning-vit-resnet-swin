The source code in this folder was, in substance, written by a large language model (config/ data/ src/).
The team's role was to decide what to build and  what to study:
  - the research question
  - experimental design
  - the policy for selecting hyperparameters (gridsearch guided/anchored to literature baselines)
  - choice of architectures and continual-learning methods
  - the class-IL evaluation protocol

  We reviewed the generated code and verified models outputs.
  Interpretation of the results was done by the team.
